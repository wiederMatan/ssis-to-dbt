## Installation

### Prerequisites

- Python 3.9+
- Node.js 18+ (for dashboard)
- dbt-core 1.5+
- SQL Server connection (for dbt execution)

### Setup

```bash
# Clone the repository
git clone https://github.com/your-org/ssis-to-dbt.git
cd ssis-to-dbt

# Install Python dependencies
pip install -r requirements.txt

# Install dbt packages
cd dbt_project
dbt deps
cd ..

# Install UI dependencies (optional)
cd ui
npm install
cd ..
```

---

## Usage

### 1. Parse SSIS Packages

```bash
# Parse all .dtsx files in a directory
python3 -m src.parser.ssis_parser ./samples/ssis_packages -o ./output -v

# Output files generated:
#   - output/parsed_packages.json
#   - output/schema_metadata.json
#   - output/parsing_report.md
```

### 2. Run Migration Pipeline

```bash
# Full migration with validation
python3 run_migration.py

# Skip dbt execution (parse only)
python3 run_migration.py --skip-dbt

# Skip validation
python3 run_migration.py --skip-validation

# Verbose output
python3 run_migration.py -v
```

### 3. Run dbt Models

```bash
cd dbt_project

# Install dependencies
dbt deps

# Run all models
dbt run

# Run tests
dbt test

# Run and test together
dbt build
```

### 4. Launch Monitoring Dashboard

```bash
cd ui
npm run dev
# Open http://localhost:5173
```

## Validation Checks

The migration validator performs three types of checks:

| Check Type       | Description                                    | Pass Criteria        |
|------------------|------------------------------------------------|----------------------|
| Row Count        | Compare legacy table vs dbt model row counts   | Variance < 0.1%      |
| Primary Key      | Check for NULL and duplicate primary keys      | 0 nulls, 0 duplicates|
| Numeric Checksum | Compare SUM/AVG of numeric columns             | Variance < 0.01%     |

---

## Manual Review Items

Some SSIS components require manual review and cannot be auto-converted:

| Component Type    | Reason                                    | Recommendation              |
|-------------------|-------------------------------------------|-----------------------------|
| Script Task       | Contains custom C#/VB.NET code            | Convert to dbt macro/Python |
| Send Mail Task    | Email notification                        | Use external alerting       |
| FTP Task          | File transfer operations                  | Use dbt external tables     |
| Execute Process   | External process execution                | Orchestration layer         |
| File System Task  | File operations                           | Pre/post hooks              |

## Security

- **XXE Protection**: The SSIS parser uses hardened XML parsing with external entity resolution disabled
- **No Network Access**: XML parser configured with `no_network=True`
- **DTD Validation Disabled**: Prevents DTD-based attacks
